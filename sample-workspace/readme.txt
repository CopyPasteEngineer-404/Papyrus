PAPYRUS - OFFLINE DOCUMENT TRANSFORMATION

Version 0.1.0

DESCRIPTION
Papyrus is a desktop application that transforms documents between different file formats. It runs entirely offline, ensuring your data never leaves your machine.

SUPPORTED INPUT FORMATS
- Markdown (.md) - Full syntax including tables, code blocks, and mermaid diagrams
- CSV (.csv) - Comma-separated values with professional parsing
- Mermaid (.mmd, .mermaid) - Diagram definition files
- Plain Text (.txt) - Standard text documents

SUPPORTED OUTPUT FORMATS
- PDF - Styled documents with proper layout and rendered diagrams
- HTML - Self-contained web pages with embedded styling
- DOCX - Microsoft Word compatible documents
- CSV - Extracted table data in spreadsheet format
- Markdown - Clean markdown output
- Plain Text - Simple text with formatting stripped

HOW TO USE
1. Open the application
2. Click "Open Workspace" to select a folder with your documents
3. Browse and select files from the workspace tree or file grid
4. Use the right panel to choose your output format
5. Click "Export" or "Convert" to generate your output

THEMES
Papyrus comes with four distinct visual themes:
- Papyrus: Classic manuscript style with gold accents
- Halftone: Retro newsprint aesthetic with dotted patterns
- Isometric: Professional 3D depth with clean lines
- Minimal Art: Gallery-inspired minimalism

SYSTEM REQUIREMENTS
- Windows 10+, macOS 10.15+, or Linux (x64)
- 4GB RAM minimum
- 200MB disk space
